- how to get indicator-brightness:

    sudo add-apt-repository ppa:indicator-brightness/ppa
    sudo apt-get update
    sudo apt-get install indicator-brightness
    
- paste icon in:

    /usr/share/notify-osd/icons/gnome/scalable/status/

- reload wingpanel

    killall wingpanel
